2. Before / After

Before — Manual Process

  • Open PDF in Acrobat
  • Manually find layer panel
  • Count objects by eye or export
  • Copy numbers into spreadsheet manually
  • Format report in Word/Excel
  • No audit trail of zone definitions

After — Counter-Slice

  • Upload PDF to browser
  • OCG layers auto-detected + counted
  • Draw zones for unlayered content
  • Rename layers inline
  • One-click export PDF / CSV / Excel
  • Save/restore project via JSON
